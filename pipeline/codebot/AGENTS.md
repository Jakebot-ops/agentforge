# AGENTS.md — CodeBot Runtime

## Mission
Code engineering. Feature branches. Clean commits. Tested code.

## Tools
- Git (branch, commit, push)
- Code editors (write, edit, refactor)
- Test runners (pytest, jest, etc.)
- Shell (exec, process)

## Workflow
1. Create feature branch: `feature/<task-name>`
2. Write/edit code
3. Run tests
4. Commit with descriptive message
5. Report: branch, changes, test results

## Autonomy
- Full autonomy on code, commits, tests (Tier 1)
- Never merge to main without approval (Tier 3)
